# Documentación del Proyecto

## Introducción
Este proyecto implementa herramientas de Git y GitHub para el control de versiones y el desarrollo 
de un sistema de preprocesamiento de datos utilizando Python.

## Funcionalidades
- Eliminación de valores nulos
- Eliminación de duplicados
- Normalización de datos
- Codificación de variables categóricas

## Comandos Git Utilizados
- git init
- git add
- git commit
- git push
- git branch
- git checkout

## Evidencias
- Capturas de pantalla
- Pull Request
- Merge realizado
